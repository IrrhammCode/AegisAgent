## 🛠️ Technical Cheat Sheet (If Judges Ask)
- **What is being proved in ZK?** The integrity of the file hash and the timestamp of discovery without revealing the actual file content.
- **Why Hypercerts?** We use them as "Impact Receipts" (ERC-1155) because they are fractionable and designed specifically to value positive impact — in this case, the impact of securing digital IP.
- **Where is the data stored?** Permanently on Arweave, ensuring it can never be censored or deleted.
